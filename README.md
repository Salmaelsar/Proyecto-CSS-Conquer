# Proyecto-CSS-Conquer

Este proyecto consiste en el desarrollo de un sitio web para una academia ficticia denominada Conquer.  
La web se estructura en diferentes páginas HTML que representan las distintas secciones del sitio, como inicio, quiénes somos, cursos, blog, registro, inicio de sesión y contacto.

El objetivo principal del proyecto es aplicar los conocimientos adquiridos en HTML y CSS, haciendo uso correcto de la semántica, la jerarquía de contenidos, la correcta organización de archivos, el uso de enlaces relativos y la inclusión de etiquetas básicas de SEO, priorizando la correcta estructuración de la información frente al diseño visual.
